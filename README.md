# Project Title
Image Stegnography

## Project Structure
- `src/`: Contains main source code files.
- `tests/`: Contains unit tests for the project.
- `docs/`: Documentation files explaining project usage and structure.

## Usage Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/harsh23csec11sist-cmd/Image_Stegnography.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Image_Stegnography
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Folder Explanations
- The `src/` folder includes core functionalities and algorithms for image steganography.
- The `tests/` folder has tests ensuring the reliability of different components.
- The `docs/` folder contains this README file along with additional documentation for users.

## Project Highlights
- Utilizes advanced algorithms for image encoding and decoding.
- Easy to use, with clear instructions provided for new users.
- Actively maintained and updated to ensure compatibility with latest technologies.

# Image Steganography

Image Steganography is a project that allows you to hide information within images. This can be useful for secure communication, watermarking, or simply for fun!.

## Usage
1. Clone the repository.
2. Install the required dependencies.
3. Run the GUI application to start embedding or extracting messages from images.

## Dependencies
- Python 3.x
- Flask
- OpenCV
- NumPy

Refer to the respective files in the repository for detailed instructions.
